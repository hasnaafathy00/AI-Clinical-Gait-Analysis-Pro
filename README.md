# 🩺 AI-Powered Clinical Gait Analysis Pro

An advanced diagnostic tool designed for physical therapists and researchers to perform automated, bilateral gait analysis using Computer Vision (Mediapipe) and Python.

## 🚀 Key Features
- **Bilateral Kinematics:** Real-time tracking of Hip, Knee, and Ankle angles for both legs simultaneously.
- **Gait Phase Detection:** Automatically identifies gait cycles (Heel Strike, Swing, and Stance phases).
- **Symmetry Scoring:** Provides a precise percentage-based symmetry score (Current Test: **99.85%**).
- **Clinical Reporting:** Generates automated Master Reports (.txt) and high-resolution kinematic graphs (.png).
- **Visual Biofeedback:** Real-time skeleton overlay with color-coded joint data.

## 📊 Sample Analysis Results
The system generates a 3x2 bilateral comparison grid to visualize abnormalities and deviations from normal thresholds.

## 🛠️ Technology Stack
- **Python 3.x**
- **MediaPipe:** For high-fidelity pose estimation.
- **OpenCV:** For real-time video processing.
- **Matplotlib & Pandas:** For data visualization and statistical analysis.
